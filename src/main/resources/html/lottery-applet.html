Java Lottery Number Generator

If you are reading this, your browser is not Java enabled.

A quick, easy, and truly random way to pick your lottery numbers - let this applet pick them for you! I wrote this Java applet as an exercise in simple animation, sorting, etc. It will pick your lottery numbers at the push of a button, so you don't have to resort to using the birthday of your budgerigar, or whatever.
